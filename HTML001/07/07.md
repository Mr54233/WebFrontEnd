## CSS 布局的三种机制
### 网页布局的核心——————就是用CSS来摆放盒子。
+ CSS 提供了3种机制来设置盒子的摆放位置,分别是**普通流(标准流)、浮动
和定位**,其中: 
+ 普通流(标准流
块级元素会独占一行,从上向下顺序排列

    `常用元素:div、hr、p、h1~h6、ul、ol、dl、form、table 行内元素会按照顺序,从左到右顺序排列,碰到父元素边缘则自动换行; 常用元素:span、a、i、img等`
浮动
让盒子从普通流中浮起来,主要作用让多个块级盒子一行显示。
定位
将盒子“定”在浏览器的某一个位置——CSS 离不开定位,特别是后面的 js 特 效。
