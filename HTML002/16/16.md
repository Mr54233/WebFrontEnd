# 构造函数和原型
## 对象的三种创建方式-- 复习
1. 字面量方式
```js
var obj = {};
```
2. new关键字
```js
var obj = new Object{};
```
3. 构造函数方式
```js
function Person(name,age){
    this.name = name;
    this.age = age;
}
var obj = new Person('zs',12)
```
