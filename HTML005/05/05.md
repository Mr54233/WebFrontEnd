# 在 JSX 使用 JavaScript 表达式

## 嵌入 js 表达式

- 数据储存在 js 中
- 语法：{JavaScript 表达式}
- 注意：语法中是单大括号，不是双大括号

```js
const name = "Tom";
const dv = (
	<div>
		<h1>你好我叫：{name}</h1>
	</div>
);
```

## 注意点

- 单大括号中可以使用任意 JavaScript 表达式

```js
const dv = (
	<div>
		<h1>{1}</h1>
		<h1>{"a"}</h1>
	</div>
);
```

- jsx 自身也是 js 表达式

```js
const h1 = <h1>我是jsx</h1>;
const dv = <div>嵌入jsx：{h1}</div>;
```

- 注意：js 中的对象是个例外，一般只会出现在 style 属性中

```js
const obj = <p>{{ a: "1" }}</p>;//这样写会报错
```
- 不能在{}中使用语句，例如if或for
